## Project Overview
This project analyzes retail sales data from a Superstore to identify revenue trends, profit drivers, customer segments, and regional performance. The goal of this project is to convert raw sales data into actionable business insights using Power BI.
## Dataset
The dataset contains transactional retail data including:
- Orders and Order Dates  
- Customers and Customer Segments  
- Products, Categories, and Sub-Categories  
- Sales, Profit, and Quantity  
- Region, State, and Ship Mode  
- Payment Mode and Returns  
The raw data is provided in both CSV and Excel format.
## Tools & Technologies
- Power BI – Data modeling, DAX measures, and dashboards  
- Excel – Data inspection  
- CSV – Raw data storage  
## Key Analysis Performed
- Sales and profit trend analysis  
- Category and sub-category performance evaluation  
- Regional and state-level sales comparison  
- Customer segment contribution to revenue  
- Identification of loss-making products and high-return items  
## Dashboard Features
- KPIs for Sales, Profit, and Quantity  
- Interactive filters by Region, Category, and Segment  
- Product-level profitability insights  
- Regional performance visualization  
- Payment mode and returns analysis  
## Business Value
This dashboard enables business stakeholders to:
- Identify high-profit and loss-making products  
- Optimize inventory and pricing strategy  
- Improve regional sales performance  
- Understand customer buying behavior  
- Make data-driven operational decisions
